SmartDoc Manual

Read, Create, Distribute

SmartDoc is a powerful application that allows you to read, create, and distribute text documents using the industry standard Doc format.

Easy

SmartDoc is easy to use. If you're already using the built-in applications, you can begin reading and writing right away.

Advanced

SmartDoc also has many advanced features: Find and replace text. Rename and reorder bookmarks. Beam and download documents.

Using SmartDoc

SmartDoc has two basic views: the List view and the Document view.

The List view displays a list of the documents that are present on your device. At a minimum, the title of the document is shown. Optionally, settings icons, the document date, or the document size may be shown by altering the Preferences. Tapping on the document title will open the document.

The Document view displays the current chosen document. You may read, edit, or perform other actions here.

Compression

The Doc format handles compressed and uncompressed text. The compression algorithm reduces memory use by 30-40% with average English language text.

The compression format is asymmetric. Decompression is much faster than compression. Decompression, which is used every time you view compressed text, is nearly instantaneous.

However, compression requires intense processing, and because of the more limited processing capabilities of your device, compression may take an extraordinary amount of time.

On current devices, the average time is about 6 seconds per kilobyte of uncompressed text, or about 12 kilobytes per minute.

Every time compressed text is displayed, it is decompressed on the fly. If it is not altered, it does not need to be recompressed. However, if it is altered, it will be recompressed and stored back in its file.

Because of this, it may be prudent to work with a document as uncompressed text, and before distribution, manually recompress it. This way you will avoid the decompression/recompression cycle every time you edit text.

Searching

SmartDoc allows two types of text searching: Through the Find menu in the List view and through the Global Find (silk-screened Find button).

The Find menu only searches through the current (open) document, searching sequentially through the text.

The Global Find searches sequentially through all documents. This could take a long time if you have many or long documents. This may be enabled or disabled through the Preferences dialog.

If Global Find is disabled, you may enable individual documents to be searched with Global Find (Overide Global Find). This is done through the Info dialog or through the Global icon in the List view. This allows you to search through only documents that you wish to search through, skipping others that don't need to be searched.

Autoscrolling

You can have SmartDoc automatically scroll the document for handless reading or for basic teleprompting-type use.

Tap on the triangle that represents the "Play" button and SmartDoc will begin scrolling.

Tap on the square that appears to stop scrolling.

Use the MemoPad and ToDo buttons to increase and decrease the scrolling speed respectively.

Press the Addresses button to pause the scrolling. Press it again to unpause.

Press the Dates button to stop scrolling.

When SmartDoc reaches the end of the document, the scrolling will automatically stop.

NOTE: While SmartDoc is scrolling or paused, your device will not go into sleep mode. Be aware not to leave SmartDoc scrolling or paused if you do not intend to do so, otherwise you may unintentionally use up battery power. As soon as you stop the scrolling or close the document, your device will sleep according to the system settings.

System Text Edit Functions

SmartDoc allows you to use the System text editing functions such as copy, paste, and undo. However, the system imposes limit to the amount of text that may be put on the clipboard. Since Doc format documents may be much larger than the working memory in your device, Select All has been changed to Select Block while working in the Document view. A block may be as large as 8k of text.

NetMarks

NetMarks are essentially bookmarks for Uniform Resource Locators (URLs). Each NetMark consists of a name for the NetMark and an associated URL. These may be saved into a list. Later versions of SmartDoc will have more NetMark management capabilities.

Installation

Install the SmartDoc.prc as you would with other software by using the installation software that accompanied your device. The SmartDoc icon will appear when you tap the Application icon.

Elements Of SmartDoc

Below is a description of all the elements available in SmartDoc.

List View

This view shows the list of documents that are present on your device.

Icons

Action- This icon allows actions to be performed on the document. Tap on the icon to popup a list of actions.

Open- Select this to view/edit the document.

Open as Copy- Select this to duplicate the document and open it.

Append Memo- Select this to append a memo the the document. A memo selection dialog will appear. Tap the title of a memo to select it.

Change Category- Select this to change the category of the document. The Category popup in the upper right corner will popup. Select a category to assign to the document.

Compression- Select this to toggle the compression status of the document. If it is compressed already, it will be uncompressed, and vice versa. See Compression.

Duplicate- Select this to duplicate the document.

Beam- Select this to beam the document.

Delete- Select this to delete the document.

Backup- This icon indicates whether the document will be backed up on the HotSync. Tap on it to toggle the setting.

Global Find- This icon indicates whether the document will be searched with a Global Find regardless of whether the Global Find preference is set. This means that if Global Find in Preferences is off, the document will still be searched.

Private- This icon indicates whether the document is a private document. Of course, if private documents are set to be hidden in the System Security preferences, the document will not appear in the list.

New Button- Tap this to create a new blank document.

Category Popup- This indicates which category to display in the List view. Tap it to popup a list of categories. Tap on one to select a category.

Book Menu

New- Select this to create a new blank document. The document will be compressed or uncompressed according to the preference selected in Preferences.

Set All Docs To Backup- Select this to set all documents to be backed up on the next HotSync.

Set No Docs To Backup- Select this to set no documents to be backed up on the next HotSync.

Set All Docs To Read Only- Select this to set all documents to Read Only.

Set All Docs To Editable- Select this to set all documents to Editable.

Beam All In Category- Select this to beam all the documents in the current category.

Download- Select this to open up the Get File dialog.

Delete All In Category- Select this to delete all the documents in the current category.

View Menu

By Name- Select this to display documents in alphabetical order.

By Size- Select this to display documents in order of increasing size.

By Date Created- Select this to display documents in order of the creation date.

By Date Last Touched- Select this to display documents in the order of last used.

Options- Select this to open up the View Options dialog.

Options Menu

About- Select this to open up the About dialog.

Beam SmartDoc- Select this to beam the SmartDoc application to another device.

Validate Code- Select this to open up the Validate Code dialog.

Preferences- Select this to open up the Preferences dialog.

Date- This indicates the date of the document. Its settings are controlled in the View Options dialog.

Size- This indicates the size of the document. If the typeface is in bold lettering, the document is uncompressed. If the typeface is in regular lettering, the document is compressed.

Document View

This view allows you to read and edit the document.

Editable/Read Only Switch- This indicates whether the document is editable. Tap it to toggle from Read Only to Editable and vice versa.

Category Popup- This indicates the category that is assigned to the document. Tap it to popup a list of categories. Tap on one to select a category.

Done Button- Tap this to close the current document and return to the List view.

Info Button- Tap this to open up the Info dialog.

F Button- Tap this to open up the Find dialog.

Play Button- Tap this to begin autoscrolling. Tap it again to stop.

Font Buttons- Tap on one of these to change the display font of the text.

Bookmark Popup- Tap this to popup a list of bookmarks for the document. Select a name to go to the position of the bookmark in the document.

Add Bookmark- Select this to bookmark the current position. The first 15 characters that appear at the top of the document will appear as the bookmark name.

Edit Bookmarks- Select this to open the Edit Bookmarks dialog.

Doc Menu

New- Select this to close the current document and create a new blank document.

Close- Select this to close the current document and return to the list view.

Info- Select this to open the info dialog.

Rename- Select this to open the info dialog. There you may rename the document.

Compression- Select this to toggle the compression status of the document. If it is compressed already, it will be uncompressed, and vice versa.

Duplicate- Select this to duplicate the document. The original document will be closed and the copy will be opened.

Prepare for Distribution- Select this to open the prepare for distribution dialog.

Upload by FTP- Select this to open the upload dialog.

Beam- Select this to beam the document.

Delete- Select this to delete the document.

Navigate menu

Add Bookmark- Select this to bookmark the current position.

Edit Bookmarks- Select this to open the edit bookmarks dialog.

Scan Embedded Bookmarks- Select this to scan the current document for embedded (bracketed) bookmarks.

Scan Custom Bookmarks- Select this to open the scan custom bookmarks dialog.

Delete All Bookmarks- Select this to delete all bookmarks.

Find- Select this to open the find dialog. This is different than global find.

Find Next- Select this to find the next instance of the last find text.

Replace/Find- Select this to open the find and replace dialog

Replace- Select this to replace with or insert the last replace text.

Go to Top- Select this to go to the top of the document.

Go to Bottom- Select this to go to the bottom of the document. Screen- Tap the screen to scroll the text. If you tap the top of the screen, the text scrolls up. If you tap the bottom of the screen, the text scrolls down. You may hold the stylus down for continuous scrolling. With continuous scrolling, the middle of the screen allows scrolling to pause.

View Options

Show Sizes- Check this to display sizes for each document in the list view. Uncheck to not display.

Show Bytes- Check this to display sizes in bytes. Uncheck this to display sizes in kilobytes (k).

Show Dates- Check this to display dates for each document in the list view. Uncheck to not display.

Created/Last- Select a display type for the dates. Select created for creation date. Select last for last date used.

Icons

Show action- Check this to display the action icon. This allows actions to be performed on the document.

Show Backup- Check this to display the Backup icon. This indicates whether the document will be backed up on the next HotSync.

Show Global- Check this to display the Global Find icon. This indicates whether the document will be searched by Global Find regardless of the Global Find setting in Preferences.

Show Private- Check this to display the Private icon. This indicates whether the document is set as a private.

Preferences

Scroll/Taps

Page/Line- Select whether to scroll by page or single line.

Overlap Line- Check this to have an overlapping line when scrolling by page.

Down Only- Check this to have the screen scroll down only when tapped.

Left Handed- Check this to have scroll bars appear on the left side.

Text

Left/Right- Select whether to left or right justify the text.

Lined Text- Check this to have lined text.

Start New Docs Compressed- Check this to have new documents created as compressed. Uncheck this to have new documents created as uncompressed.

Autoscan Embedded Bookmarks- Check this to have uninitialized documents automatically scanned for embedded (bracketed) bookmarks.

Allow Global Find- Check this to enable Global Find for SmartDoc. Uncheck to disable Global Find for SmartDoc.

Prepare For Distribution

Go To Top Of Document- Check this if you wish to reset the document to the top.

Backup On HotSync- Check this if you wish to have the document backed up on the next HotSync.

Undo Private (If Set)- Check this if you wish to undo the Private setting for the document.

Undo Overide Global Find (If Set)- Check this if you wish to undo the Overide Global Find setting for the document.

Set Category To Unfiled- Check this if you wish to set the document to the Unfiled category.

Compress (If Not Compressed)- Check this if you wish to compress the document. Compression may take a long time to perform. See Compression.

Default Button- Tap this to set all the check boxes to the recommended actions.

Ok Button- Tap this to perform the chosen actions.

Edit Bookmarks

Bookmark List- The list of bookmarks available for the current document will appear. To select a bookmark, tap on it and a flashing insertion point will appear. You may then edit the name of the bookmark. You may also change the order of a bookmark by dragging and dropping to another location on the list.

Go Button- Tap this to go to the selected bookmark. If none is selected, this will do nothing.

Delete Button- Tap this to delete the selected bookmark. If none is selected, this will do nothing.

Delete All Button- Tap this to delete all bookmarks in the list.

Scan Embedded Button- Tap this to scan the current document for embedded (bracketed) bookmarks.

Scan Custom Button- Tap this to open the Scan Custom Bookmarks dialog.

Sort Alphabet Button- Tap this to sort the bookmarks in alphabetical order.

Sort Appear Button- Tap this to sort the bookmarks in order of appearance in the document.

Scan Custom Bookmarks

Search For Field- Enter the text for the bookmark search.

Only At Beginning Of Line Check Box- Check this if the text should only be found at the beginning of a line of text.

Bookmark Names

Name Check Box- Check this if every bookmark has the same name. "Chapter" would be a good example.

Name Field- Enter the name of the bookmark. "Chapter" would be a good example.

Add Count Check Box- Check this if you wish to automatically number the bookmarks.

Number Of Characters Check Box- Check this if you wish to include the text after the search text to be used in the bookmark name.

Number Of Characters Field- Enter the number of characters to include. There is a limit of 15 characters.

Find And Replace

Text To Look For- Enter the text to search.

Text To Replace With- Enter the text to replace with.

Find Button- Tap this to find the next instance of the search text.

Replace Button- Tap this to replace the selected text with the replacement text.

R/F Button- Tap this to replace the selected text with the replacement text and find the next instance of the search text.

Get File

Netmark Name Field- Enter the name of the NetMark. A description of the URL would be appropriate.

User Netmarks Popup- Tap this to popup a list of NetMarks that may be maintained by you.

URL Field- Enter the URL to get.

Text Buttons- Tap one of these buttons to insert/replace the corresponding text into the URL.

History Popup- Tap this to popup a list of previously used NetMarks. Select one to use it.

User Field- Enter the user name if accessing an FTP site. You may leave it blank if you are using an anonymous FTP site.

Pass Field- Enter the password if accessing an FTP site. You may leave it blank if you are using an anonymous FTP site.

Add Button- Tap this to add the current NetMark to the User NetMarks list.

Clear Button- Tap this to clear the User NetMarks list.

Get Button- Tap this to retrieve the URL from the Internet. Your device must be set up properly to access the Internet. See the users manual that came with your device for more details.

Upload File

Netmark Name Field- Enter the name of the NetMark. A description of the URL would be appropriate.

URL Field- Enter the URL to get.

Text Buttons- Tap one of these buttons to insert/replace the corresponding text into the URL.

History Popup- Tap this to popup a list of previously used NetMarks. Select one to use it.

User Field- Enter the user name if accessing an FTP site. You may leave it blank if you are using an anonymous FTP site.

Pass Field- Enter the password if accessing an FTP site. You may leave it blank if you are using an anonymous FTP site.

Save As Text File Check Box- Check this if you are sending the file as a text file. Leave it unchecked if you are sending the file as a Palm database (pdb file).

Put Button- Tap this to send the document to the URL on the Internet. Your device must be set up properly to access the Internet. See the users manual that came with your device for more details.

Info

Title Field- This displays the title of the document. If you wish to change it, edit the text.

Category Popup- This indicates the category that is assigned to the document. Tap it to popup a list of categories. Tap on one to select a category.

Size- This indicates the size of the document in bytes. This disregards any overhead for the database.

Compression Status- This indicates the compression status of the document.

Private Check Box- Check this if you wish the document to be private. Uncheck it if you don't. Private documents will not be searched with Global Find. Remember that Private documents are not "secure" documents.

Backup Check Box- Check this if you wish the document to be backed up on the next HotSync. Uncheck it if you don't.

Overide Global Find Check Box- Check this if you wish to have the document searched by Global Find regardless of the Global Find setting in Preferences.

Delete Button- Tap this to delete the document.

Validate Code

Regcode Field- This displays the RegCode of your device. It is based on your HotSync ID.

Name- Enter your name here.

Organization- Enter the name of your organization here.

Validation Code- Enter the validation code that you received after purchasing SmartDoc.

Ok- Tap this to validate.

Edit Menu

Undo- Select this to undo the last Edit menu function.

Cut- Select this to cut the selected text.

Copy- Select this to copy the selected text.

Paste- Select this to paste previously cut or copied text.

Select All- Select this to select all the text.

Select Block- Select this to select a block of text. Since only a portion of a document is used/visible at a time, only a block will be selected.

Keyboard- Select this to bring up the System keyboard.

Graffiti- Select this to bring up the Graffiti help.
